# Potato Wi-Fi Attack demonstration

This project is designed to establish a controlled laboratory environment for demonstrating Wi-Fi security vulnerabilities to students and visiting high schoolers.
The primary goal is to provide a practical, visual understanding of how wireless communication can be compromised, and more importantly, how to identify and prevent such risks. 
By using local resources and isolated devices, the infrastructure ensures a safe space to connect user behaviour with technical network monitoring.

## Setup
In order to use the Potato Wi-Fi attack demonstration, you will need the following devices:
- A Raspberry PI, or any other device that is capable of hosting a Wi-Fi network.
- A Linux device for running the attack demonstrations.
- A Wi-Fi antenna that is capable of running in monitor mode.

Details on how to setup and run the project are provided in the ``docs/`` folder. 

## Project Structure

``attacker-scripts/`` - Scripts for the demonstrator (attacker) device
``router-scripts/`` - Scripts for the router (victim) device
``docs/`` - Documentation
